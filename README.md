# Репозиторий для обработки юридических данных

## Обзор
Этот репозиторий предназначен для сбора, обработки и анализа юридических данных. Он включает инструменты для сбора данных, проведения разведочного анализа данных (EDA) и хранения необработанных данных.

## Структура репозитория
- `agents/`
  - Содержит скрипт `data_collection_agent.py` для сбора данных из различных источников.
- `data/`
  - `raw/`
    - Содержит файл необработанных данных:
      - `legal_corpus.csv`
- `notebooks/`
  - `eda.ipynb`
    - Ноутбук для разведочного анализа данных (EDA).
- `requirements.txt`
  - Список зависимостей, необходимых для проекта.
- `run.py`
  - Точка входа для запуска проекта.

## Начало работы

### Предварительные требования
- Python 3.8 или выше
- Виртуальное окружение (рекомендуется)

### Установка
1. Клонируйте репозиторий:
   ```bash
   git clone <repository_url>
   ```
2. Перейдите в директорию проекта:
   ```bash
   cd chufistov_data_for_ml
   ```
3. Создайте и активируйте виртуальное окружение:
   ```bash
   python -m venv .venv
   .\.venv\Scripts\activate
   ```
4. Установите необходимые зависимости:
   ```bash
   pip install -r requirements.txt
   ```

## Использование

### Сбор данных
Используйте скрипт `data_collection_agent.py` для сбора данных из различных источников. Пример использования:
```python
from agents.data_collection_agent import DataCollectionAgent

agent = DataCollectionAgent()
df = agent.run(
        sources=[
            {
                'type': 'kaggle_dataset',
                'dataset_name': 'athugodage/russian-legal-text-parallel-corpus',
                'file_path': 'legal_text_corpus.csv'
            },
            {
                'type': 'scrape',
                'url': 'https://base.garant.ru/12150845/',
                'name': 'ЛК РФ'
            },
        ]
    )
```

## Источники данных
- Спаршенные данные хранятся в папке `data/raw/`.
- Поддерживаемые источники данных:
  - Открытые наборы данных (kaggle).
  - Публично доступные юридические порталы, содержащие российские кодексы (АО "Гарант", Консультант Плюс).
